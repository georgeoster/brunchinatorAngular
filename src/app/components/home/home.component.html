It started as a simple tool to help me remember where I liked the bloody mary.

Then I wanted to know what my friends thought too. So I added users.

Those friends wanted to rate burgers as well as bloodys. So I added burgers.

So now you can see what other people have to say, and if you want to leave some reviews yourself you can create an account and join in on the fun.
